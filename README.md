NEOCPhelper
===========

Takes the Minor Planet Center NEOCP data and provides the data in suitable
formats for TheSkyX and Find_Orb

NEOCPHelper by Albert.white@gmail.com (Z72). Version 1.0

Loads the NEOCP list from the MPC and provides observations for Find_ORB and
also asteroid databases for TheSkyX.

In order for the 'Upadte from TheSkyX' button to populate the data you will
need to have a version of TheSkyX with the TCP server enabled and also have
previosuiosly loaded the objects in the Small Asteroid Database.
